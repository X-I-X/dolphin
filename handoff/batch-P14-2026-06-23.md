### Fix 1: Archive-notice self-referential href (all 19 pages)

Same defect pattern as P11/P12/P13. The archive-notice anchor's `href` pointed back to the local filename rather than to johnclilly.com.

**Before:**
```html
<a href="BT_JCL_ICERC1999.html" rel="noopener" data-original-href="http://www.johnclilly.com/BT_JCL_ICERC1999.html">
```

**After:**
```html
<a href="http://www.johnclilly.com/BT_JCL_ICERC1999.html" target="_blank" rel="noopener" data-original-href="http://www.johnclilly.com/BT_JCL_ICERC1999.html">
```

Applied to all 19 pages.
