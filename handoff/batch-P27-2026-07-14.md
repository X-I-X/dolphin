## Fixes applied

### All 6 pages — Invalid HTML structure repaired

**Problem:** Every page had a malformed caption + navigation block. The original scraper/builder left this invalid structure:

```html
<p>

[caption text]

</p><p>
<p class="page-nav-return">← Nav links</p>



</p></div>
```

This is invalid HTML — a `<p>` element cannot contain a block-level `<p>` child. Browsers handle it inconsistently; some rip the inner `<p>` out, others suppress it. Causes layout glitches on some mobile browsers.

**Fix applied to all 6 pages:** Replaced with clean sibling structure:

```html
<p class="photo-caption">[caption text]</p>
<p class="page-nav-return">← Nav links</p>
</div>
```

The outer wrapper `<div style="text-align:center">` now closes cleanly after both elements.
