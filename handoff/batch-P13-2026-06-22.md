## Fixes applied

### 1. Archive-notice self-referential href fixed (20 pages)

Same defect as P11/P12. All 20 non-x stubs had the archive-notice link `href` pointing back to the local filename rather than to johnclilly.com.

**Before:**
```html
<a href="doljump01.html" rel="noopener" data-original-href="http://www.johnclilly.com/doljump01.html">
```

**After:**
```html
<a href="http://www.johnclilly.com/doljump01.html" target="_blank" rel="noopener" data-original-href="http://www.johnclilly.com/doljump01.html">
```

Pages with `&` in filename used HTML-encoded form (`JL&amp;Barbara.html` → `http://www.johnclilly.com/JL&amp;Barbara.html`) — correct HTML encoding preserved.
