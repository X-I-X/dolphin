## Fixes applied

### Fix 1: Archive-notice self-referential href (all 12 pages)

Identical systemic defect as P11–P17. The `href` attribute in the archive-notice anchor pointed to the local filename instead of the live johnclilly.com URL.

**Before (example — lillywave.html):**
```html
<a href="lillywave.html" rel="noopener" data-original-href="http://www.johnclilly.com/lillywave.html">
```

**After:**
```html
<a href="http://www.johnclilly.com/lillywave.html" target="_blank" rel="noopener" data-original-href="http://www.johnclilly.com/lillywave.html">
```

Applied to all 12 pages.
