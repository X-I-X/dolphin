### Fix 1: Archive-notice self-referential href (australia01–04, brain, brain02)

Identical defect pattern as P11–P15. The `href` attribute on the archive-notice anchor pointed to the local filename instead of the live johnclilly.com URL.

**Before (example — australia01.html):**
```html
<a href="australia01.html" rel="noopener" data-original-href="http://www.johnclilly.com/australia01.html">
```

**After:**
```html
<a href="http://www.johnclilly.com/australia01.html" target="_blank" rel="noopener" data-original-href="http://www.johnclilly.com/australia01.html">
```

Applied to: australia01.html, australia02.html, australia03.html, australia04.html, brain.html, brain02.html.
